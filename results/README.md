# Results

Generated by the Phase 2–4 notebook. These are the exact values reported in the
paper.

## `tables/` — paper tables 1–6

| File | Content |
|---|---|
| `table_1_test_accuracy_by_method_protocol.csv` | Test accuracy (mean ± std) per method × protocol. |
| `table_2_macro_f1_by_method_protocol.csv` | Macro F1 (mean ± std) per method × protocol. |
| `table_3_per_class_f1_by_protocol.csv` | Per-class F1 (DenseNet201 + TTA + mixup) across protocols. |
| `table_4_statistical_tests_summary.csv` | Paired comparisons: mean diff, Cohen's d, Wilcoxon p, 95% bootstrap CI. |
| `table_5_literature_benchmark_comparison.csv` | Comparison against prior reported results. |
| `table_6_ensemble_strategy_distribution.csv` | Selected ensemble fusion rule per protocol × seed. |

## `summaries/` — per-seed and aggregate detail

- `DenseNet201_all_scenarios_final_summary.csv` — all protocols/seeds/methods.
- `CP4_ensemble_seed_level.csv`, `CP4_ensemble_aggregate_by_protocol.csv`,
  `CP4_final_paper_table_all_protocols.csv` — ensemble results.
- `DenseNet201_*_paired_delta_final.csv` — paired per-seed deltas between
  protocols/methods.
- `statistical_tests_results.csv` — full statistical test outputs.
- `per_class_f1_aggregate.csv` — per-class F1 for all methods/protocols.
- `final_split_size_summary.csv`, `final_class_count_summary.csv` — dataset
  composition.
- `final_interpretation_summary.json` — narrative interpretation of results.
